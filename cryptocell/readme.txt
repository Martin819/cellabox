How to build the ARMCryptoCell (libmbedcrypto-cc310.a) for nRF52840:
1.) Install make-3.81.exe
2.) Change to folder: <sdk-install-folder>/components/thread/experimental/crypto/nrf52_armgcc/armgcc
3.) Run make
4.) Copy the libmbedcrypto-cc310.a library into the OpenThread folder: <sdk-install-folder>/external/openthread/lib/gcc
